        <section class="main-section" id="Introduction">
            <header>Introduction</header>
            <p>CSS Grid Layout is the most powerful layout system available in CSS. It is a 2D system, meaning it can handle both columns and rows.</p>
            <p>Unlike flexbox, which is largely 1D, Grid allows for much more complex designs without nesting multiple containers.</p>
            <code>display: grid;</code>
        </section>

        <section class="main-section" id="Grid_Containers">
            <header>Grid Containers</header>
            <p>To get started you have to define a container element as a grid with display: grid.</p>
            <p>Once you do this, you can set column and row sizes using various properties:</p>
            <ul>
                <li>grid-template-columns</li>
                <li>grid-template-rows</li>
                <li>grid-gap</li>
                <li>justify-items</li>
                <li>align-items</li>
            </ul>
            <code>grid-template-columns: 200px auto;</code>
        </section>

        <section class="main-section" id="Grid_Items">
            <header>Grid Items</header>
            <p>The direct children of a grid container automatically become grid items.</p>
            <p>You can control their position using line numbers or named areas.</p>
            <code>grid-column: 1 / 3;</code>
            <p>This allows items to span across multiple columns or rows easily.</p>
        </section>

        <section class="main-section" id="Fractional_Units">
            <header>Fractional Units</header>
            <p>Grid introduced a new length unit called the fractional unit (fr).</p>
            <p>One fr represents a fraction of the free space in the grid container.</p>
            <code>grid-template-columns: 1fr 2fr 1fr;</code>
            <p>This creates three columns where the middle one is twice as wide as the others.</p>
        </section>

        <section class="main-section" id="Browser_Support">
            <header>Browser Support</header>
            <p>Modern browser support for CSS Grid is excellent, covering over 95% of users globally.</p>
            <p>For older browsers, you can use feature queries to provide fallbacks:</p>
            <code>@supports (display: grid) { ... }</code>
            <p>Always test your layouts in multiple environments to ensure accessibility.</p>
        </section>
    </main>
